# VS Code 使用技巧和常用快捷键



## 常用

| 快捷鍵 | 作用 | 
| --------- | ----| 
| <code>Ctrl</code>+<code>P</code>                      | 最近打开文件  | 
| <code>Ctrl</code>+<code>Shift</code>+<code>P</code>   | 显示命令菜单	| 
| <code>Ctrl</code>+<code>,</code>                      | 打开设置			| 


## 编辑

| 快捷鍵 | 作用 | 
| ------------------------------------------------------ | ---------------| 
| <code>Alt</code>+<code>↑</code>/<code>↓</code>         | 移动当前行     |
| <code>Alt</code>+<code>Shift</code>+<code>↑</code>/<code>↓</code>  | 复制当前行     | 
| <code>Ctrl</code>+<code>Shift</code>+<code>Del</code>  | 删除当前行     | 
| <code>Ctrl</code>+<code>Shift</code>+<code>Del</code>  | 删除当前行     | 
| <code>Ctrl</code>+<code>Enter</code>                   | 在当前行下方插入新的一行 |  
| <code>Ctrl</code>+<code>Shift</code>+<code>Enter</code>| 在当前行上方插入新的一行 |  
| <code>Ctrl</code>+<code>\[</code>/<code>\]</code>      | 控制当前行缩进  | 


## 多行操作

| 快捷鍵 | 作用 | 
| ------------------------------------------------------ | ---------------| 
| <code>Ctrl</code>+<code>Click</code>                    | 在点击位置插入光标           |
| <code>Ctrl</code>+<code>U</code>                        | 撤销最后一次光标插入操作     |
| <code>Ctrl</code>+<code>Alt</code>+<code>↑</code>/<code>↓</code>  | 在上下方插入光标     | 
| <code>Shift</code>+<code>Alt</code>+<code>L</code>      | 在所有选中行结尾插入光标     | 
| <code>Shift</code>+<code>Alt</code>+<code>拖动光标</code>| 同时在多个行插入光标     | 

## 代码折叠


| 快捷鍵 | 作用 | 
| --------------------------------------------------------------------------  | ---------------------- | 
| <code>Ctrl</code>+<code>Shift</code>+<code>\[</code>/<code>\]</code>        | 折叠/展开当前区域代码   | 
| <code>Ctrl</code>+<code>K</code> +  <code>Ctrl</code> +<code>\[</code>/<code>\]</code>    | 折叠/展开当前区域和子区域代码     | 
| <code>Ctrl</code>+<code>K</code> +  <code>Ctrl</code> +<code>0[</code>/<code>j</code>    | 折叠/展开所有区域代码     | 


## 导航


| 快捷鍵 | 作用 | 
| ------------------------------------------------------ | ---------------| 
| <code>Ctrl</code>+<code>G</code>                        | 跳转到指定行     |
| <code>Ctrl</code>+<code>T</code>                        | 搜索所有符号并跳转|
| <code>Ctrl</code>+<code>Shift</code>+<code>O</code>     | 复制到符号处     | 
| <code>Ctrl</code>+<code>Shift</code>+<code>M</code>     | 打开问题展示面板 | 
| <code>Ctrl</code>+<code>Shift</code>+<code>Del</code>  | 删除当前行     | 
| <code>Ctrl</code>+<code>Shift</code>+<code>Del</code>  | 删除当前行     | 
| <code>Ctrl</code>+<code>Enter</code>                   | 在当前行下方插入新的一行 |  
| <code>Ctrl</code>+<code>Shift</code>+<code>Enter</code>| 在当前行上方插入新的一行 |  
| <code>Ctrl</code>+<code>\[</code>/<code>\]</code>      | 控制当前行缩进  |
